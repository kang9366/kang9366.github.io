### 행렬의 연산
1. A + B = B + A -> 덧셈의 교환법칙
2. (A + B) + C = A + (B + C) -> 덧셈의 결합법칙
3. A + O  = O + A -> 덧셈의 항등법칙
4. A + (-A) = (-A) + A = O -> 덧셈의 역원
5. c(A + B) = cA + cB -> 스칼라 곱의 배분법칙
6. (c + d)A = cA + cB -> 스칼라 곱의 배분법칙
7. AB ≠ BA -> 행렬의 곱셈에서는 교환법칙이 성립하지 않는다.
8. (AB)C = A(BC) -> 행렬의 곱셈에서 결합법칙은 성립한다.  

#### 행렬의 거듭제곱
A가 정방행렬이고 k가 양의 정수라고 할 때 A^k는 A를 k번 곱한 것이다.
